# Magni
**This repository has been deprecated. Use [magfy](https://github.com/waneon/magfy) instead.**

Magni is a simple, fast and smooth windows magnifer

## Use

* `Alt+1` toggle magnifier
* `Alt+2` exit program

## Build

```bash
cargo build --release
```

### Build dependencies

* msvc
* rust-msvc toolchain
